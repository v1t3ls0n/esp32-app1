# ⁧🧑‍🏫 מערך שיעור למורה — Claude Code, Git/GitHub, Vercel ו-ESP32⁩

> ⁧90 דקות · לתלמידים ללא רקע · מבוסס על [המדריך המלא](../README.md)⁩

## ⁧🎯 מטרות השיעור⁩

⁧בסוף השיעור התלמידים יידעו:⁩

1. ⁧להסביר במשפט אחד מה זה שרת, ענן, API, ריפו ודיפלוימנט⁩
2. ⁧לתאר את שיטת העבודה: בקשה לקלוד ← קוד ← קומיט ← דיפלוימנט אוטומטי⁩
3. ⁧להתקין PWA עם נוטיפיקיישנים בטלפון שלהם⁩
4. ⁧לצרוב ESP32 ששולח נוטיפיקיישן אמיתי דרך הענן⁩

## ⁧🎒 הכנות לפני השיעור (למורה!)⁩

- ⁧✅ לוודא שהמערכת שלכם עובדת מקצה לקצה: לחיצת BOOT ← נוטיפיקיישן. **אל תגיעו לכיתה בלי דמו עובד.⁩**
- ⁧✅ להכין נקודת גישה (Hotspot) מהטלפון — ראו "מלכודות כיתה" למטה⁩
- ⁧✅ לוודא לכל תלמיד/זוג: חשבון GitHub, גישה ל-Claude, לוח ESP32 + כבל⁩
- ⁧✅ להקרין את המדריך (README) — הוא בנוי להיות חומר הלימוד עצמו⁩

## ⁧⚠️ מלכודות כיתה — מניסיון⁩

| ⁧מלכודת⁩ | ⁧פתרון⁩ |
|---|---|
| ⁧**WiFi בית-ספרי חוסם** התקני IoT / דורש דף התחברות⁩ | ⁧ה-ESP32 לא יודע להתמודד עם דפי התחברות. הפתרון: **Hotspot מטלפון** (זכרו: 2.4GHz בלבד — באנדרואיד לוודא שה-Hotspot לא במצב 5GHz)⁩ |
| ⁧חצי מהכיתה נתקעת באותו שלב⁩ | ⁧עצרו את כולם והראו על המקרן. אם רק זוג אחד — שלחו אותם לנוסחת ח.1 מול קלוד לבד קודם⁩ |
| ⁧קצב לא אחיד — זוגות שמסיימים מהר⁩ | ⁧יש להם את התרגול (ח.5): תרגיל 1 מיידי, תרגיל 3 לחזקים⁩ |
| ⁧תלמיד בלי אנדרואיד (אייפון)⁩ | ⁧Web Push עובד באייפון רק אחרי "הוספה למסך הבית" (iOS 16.4+). זה עובד — רק סדר הפעולות שונה: קודם להתקין, אחר-כך להפעיל נוטיפיקיישנים⁩ |
| ⁧Vercel דורש אימות טלפון בהרשמה⁩ | ⁧לבקש מהתלמידים להירשם **מראש בבית** (לשים בהודעה לפני השיעור)⁩ |
